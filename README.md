# Installation

### Set zsh as default Terminal
### Clone this Repo
```bash
git clone https://github.com/not-scripter/dotfiles.git
```
### Navigate to /dotfiles/install
```bash
cd dotfiles/install
```
### Run Installation Script
```bash
bash bootstrap.sh
```
### Install all Dependencies
#### For Termux
```bash
bash termux-deps.sh
```
