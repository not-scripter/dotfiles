# Installation

## Android

- **Termux**

```bash
git clone https://github.com/not-scripter/dotfiles.git
bash ~/dotfiles/install/bootstrap.sh
bash ~/dotfiles/install/termux-deps.sh
```

- **Nix-on-droid**

```bash
git clone https://github.com/not-scripter/dotfiles.git
bash ~/dotfiles/install/bootstrap.sh
nix-on-droid switch
```

## Linux

- **Fedora, etc**
